# SQF Score Optimisation

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/MathiasBuff/SQF-Optimisation/HEAD?urlpath=%2Fdoc%2Ftree%2FNotebook.ipynb)

This project provides a lightweight, model-based workflow to predict chromatogram behavior across method conditions and compute separation scores for method development and optimisation. It is designed to support rapid exploration of method parameters and objective-driven selection of operating points.

Work in progress.
